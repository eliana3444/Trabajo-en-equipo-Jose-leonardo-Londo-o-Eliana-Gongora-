## Preguntas y Respuestas
Pregunta: ¿Cuál es el objetivo del programa?  
Respuesta: Calcular el perímetro de un triángulo rectángulo dados los dos catetos (lados adyacentes al ángulo recto), usando el teorema de Pitágoras para la hipotenusa.

Pregunta: ¿Qué información tendrá que suministrar el usuario que ejecute el programa?  
Respuesta: La longitud de los dos catetos (como números decimales o enteros).

Pregunta: ¿Cuál es el objetivo de cada bloque?  
Respuesta:  
- Bloque de definiciones de funciones: Define `perimetro_triangulo` (calcula el perímetro sumando catetos e hipotenusa) y `calcular_hip` (calcula la hipotenusa con Pitágoras).  
- Bloque de inputs: Obtiene las longitudes de los catetos del usuario como cadenas.  
- Bloque de conversiones: Convierte las cadenas a números flotantes para cálculos.  
- Bloque de llamada a función: Ejecuta el cálculo del perímetro.  
- Bloque de print: Muestra el resultado al usuario.

Pregunta: ¿Qué es lo que primero se ejecuta?  
Respuesta: Las definiciones de las funciones (aunque no "ejecutan" en el sentido de hacer cálculos, se procesan primero). Luego, los inputs son lo primero que interactúa con el usuario y realiza acciones en tiempo de ejecución.

Pregunta: ¿Cuál es la diferencia entre lo que está escrito en español y lo que está escrito en inglés?  
Respuesta: El español se usa para textos dirigidos al usuario (inputs y outputs), haciendo el programa accesible en ese idioma. El inglés son las palabras reservadas y funciones del lenguaje Python, que son universales y no se traducen.
